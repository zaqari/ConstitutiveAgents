  Formal model of lexico-semantic constraints and word learning

We model agents as having a vocabulary consisting of tokens $w$ that can be used to refer to one or more features of speakers' social environments. Every feature is treated as continuously distributed, and we can refer to specific values of a feature $f$ at some time $t$ as $f_t$.

A word $w$ is selected on the basis of a speakers' belief about what values of $f$ a word is useful in invoking. Ergo, words are modeled as being selected via the Gaussian process below:

$$P_w(f_t) = \mathcal{N}_{w}(f_t | \mu_{f_t}, \sigma_{f_t}^2)$$

Importantly, speakers can update what values of a feature based observations of how other speakers use their own lexicons (for what values $f_t$ other speakers might use a word to refer to). Thus, a speaker's belief of what value $f_t$ a token is useful to refer to updates via the following:

$$\begin{align} \mu_{f_t} &= \mu_{f_{t-n}} + \left[ \Delta\mu_{f_t} \right] \\ \sigma^2_{f_t} &= \sigma^2_{f_{t-n}} + \left[ \Delta\sigma^2_{f_t} \right] \end{align}$$

And the values for $\left[ \Delta\mu_{f_t} \right]$ and $\left[ \Delta\sigma^2_{f_t} \right]$ are expressed as

$$\begin{align}\left[ \Delta \mu_{f_t} \right] =& \frac{f_t}{n_{f_{t-n}}+1} - \frac{\sum_n f_{t-n}}{n_{f_{t-n}}(n_{f_{t-n}} +1)} \\ \left[ \Delta \sigma^2_{f_t} \right] =& \frac{(f_t-\mu_{f_t})^2}{n_{f_{t-n}} + 1} - \frac{\sum ( f_{t-n} - \mu_{f_{t-n}})^2}{n_{f_{t-n}}(n_{f_{t-n}}+1)} + \left[ \Delta\mu_{f_t} \right]\frac{\sum (\left[ \Delta\mu_{f_t} \right] + 2 f_{t-n} - 2\mu_{f_{t-n}} ) }{n_{f_{t-n}}+1}\end{align}$$

Social Network with 1-Hot Vocabulary and occasional shuffling of all vocabulary terms

This is perhaps the most unrealistic scenario, but it demonstrates how constitutive proceses can still yield stable lexicons across agents. Agents are initialized in a random, sparsely connected social network with k connections (k=5 for the visualizations provided). Each agent has a vocabulary such that every word can refer to a single possible feature. All agents have a preference for when they might use a word to refer to a value for a given feature. Some agents have no prior, however, on specific vocabulary item-feature mappings and are free to learn them through interaction. Every so often, the agents' vocabularies are completely shuffled so that words refer to entirely new things at random.
